CryGetter

CryGetter is a computational tool that aims to perform automatic retrieval of Cry protein data.

Welcome to CryGetter Project Page!

CryGetter is a computational tool used to automaticaly retrieve Cry protein data and to offer means to analyse them. It is being developed by David Buzatto, as a part of his Ph.D. project at UNAERP (Universidade de Ribeirão Preto), with the advisement of Prof. Sônia Marli Zinagaretti. The tool was written using the Java programming language, so, to run the binary release, you will need to install a Java Virtual Machine (7.0 or higher) on your computer (click here to download the latest version).

We want to thank IFSP - Instituto Federal de Educação, Ciência e Tecnologia de São Paulo for the financial support.
